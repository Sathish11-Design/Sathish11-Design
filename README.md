### 🔷 Project 2 — AWS Resource Monitoring & Reporting Automation

> *A lightweight automation solution to monitor AWS resources, generate periodic reports, and deliver real-time change notifications via Slack.*

```
⚙️ Automation Flow:

  Cron Scheduler (every 5 min)
         │
    Bash Script + AWS CLI
         │
    ┌────┴────┐
    │         │
  EC2       S3 / Lambda / IAM
    │
  Resource Report (Timestamped)
         │
    Diff Detection (vs previous report)
         │
  Change Detected? ──► Slack Alert 🔔
         │
  Rolling 7-Report Retention 🗂️
```

| Component | Implementation |
|-----------|---------------|
| 📜 **Scripting** | Bash + AWS CLI for automated resource polling |
| 📡 **Coverage** | EC2 instances, S3 buckets, Lambda functions, IAM users |
| ⏱️ **Scheduling** | Cron-based every-5-minute automated report generation |
| 🔔 **Alerting** | Slack integration — real-time change detection notifications |
| 🗂️ **Retention** | Rolling 7-report storage to optimise cost and storage usage |
| 📈 **Impact** | Reduced manual monitoring overhead, improved cloud resource visibility |
